# Customer Churn Analysis and Prediction in the Telecommunications Sector Using Machine Learning

## Project Overview
This project analyses customer churn behaviour using a telecommunications customer dataset. The aim is to identify key factors related to churn and compare different machine learning models for churn prediction.

## Objectives
- Analyse customer churn patterns through exploratory data analysis.
- Identify important churn-related variables.
- Build and compare machine learning models.
- Provide business recommendations for customer retention.

## Dataset
The dataset used in this project is a modified version of the IBM Telco Customer Churn dataset obtained from Kaggle. It includes customer demographic information, service usage, contract type, satisfaction indicators, customer service requests, and churn status.
